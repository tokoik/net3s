## 2. 使用方法

動したらＬａｖｉｅの３Ｄボタンを押してください．スペースをタイプすればアニメーションが開始されます．マウスの左ボタンで結び目をドラッグしたり，マウスの左ボタンで図形を回転したりしてみてください．

青い結び目は「固定」されており，ほかの結び目の影響を受けません．これもドラッグして位置を変えることができます．マウスの中ボタンで，結び目の固定と開放を切り替えることができます．すべての結び目を開放してしまうと，図形が下に落下してしまいます（復旧する方法はありません）．その場合はｍまたはｐのキーをタイプして，図形を初期化してください．

ｖキーをタイプすれば，画面表示を「線画＋結び目」→「隠面消去」→「隠面消去＋結び目」→「結び目のみ」→「線画のみ」の順に切り替えます．

なお，図形は（初期値では）表示面より奥に配置しているので（そうしないとマウスカーソルが図形と重なった時に混乱してしまうので），立体感を得るには少し慣れが必要かもしれません．

### 2.1 マウス操作

- 左ボタン
  - 結び目をドラッグできます．
- 中ボタン
  - 結び目を固定したり開放したりできます．
- 右ボタン
  - ドラッグすれば図形を回転できます．

### 2.2 キー操作

- スペースバー
  - アニメーションを開始または停止します（トグル動作）．
- ｍキー
  - 図形を板状に初期化し，アニメーションを停止します．
- ｐキー
  - 図形を円筒形に初期化し，アニメーションを停止します．
- ｏキー
  - 視差を広げます．
- ｃキー
  - 視差を狭めます．
- ｎキー
  - 図形を手前に近づけます．
- ｆキー
  - 図形を遠方に遠ざけます．
- ｖキー
  - 表示方法を切り替えます．
- ESC, ｑキー
  - プログラムを終了します．

## 3. 問題点

時々，Lavie G のビデオコントローラ (ATI RADEON9600) が停止してしまうことがあります．その場合は電源スイッチの長押しでリセットしてください．
